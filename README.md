# 💎 TrueCost Pro

[![Live Demo](https://img.shields.io/badge/live%20demo-truecost--pro-gold)](https://banksiasprings.github.io/truecost-pro/)

Premium vehicle comparison calculator for Australia. Dark navy interface with gold accents, enhanced with 342 vehicle images and advanced comparison tools. Perfect for commercial and personal vehicle fleet analysis.

## ✨ Features

- **Premium UI** – Dark navy (#0B1220) with gold accents (#F59E0B)
- **342 Vehicle Images** – Wikipedia-sourced images for every vehicle (lazy-loaded)
- **Advanced Comparisons** – Side-by-side multi-vehicle analysis with detailed metrics
- **Vehicle Database** – Comprehensive Australian vehicle catalog with specifications
- **True Cost Calculation** – Depreciation, fuel, insurance, registration, servicing, tyres, stamp duty
- **State-Specific** – Registration and stamp duty calculations for all Australian states
- **EV Ready** – Special handling and incentives for electric vehicles
- **Professional Design** – Inter font, refined typography, premium appearance
- **Offline Support** – Full PWA with service worker caching
- **Responsive Design** – Optimized for desktop, tablet, and mobile
- **Version:** v1.1

## 🛠️ Tech Stack

- **Frontend:** Vanilla JavaScript (ES6+), HTML5, CSS3
- **UI Framework:** Custom CSS with premium design system
- **Images:** Wikipedia vehicle images (lazy-loaded for performance)
- **PWA:** Service Worker for offline functionality
- **Hosting:** GitHub Pages
- **Typography:** Inter font family
- **No dependencies** – Pure HTML, CSS, JavaScript

## 🚀 Getting Started

### Online
Open https://banksiasprings.github.io/truecost-pro/ in your browser to explore the premium interface.

### Development
```bash
git clone https://github.com/banksiasprings/truecost-pro.git
cd truecost-pro
# Serve the root directory with any static file server
python3 -m http.server 8000
# Open http://localhost:8000 in your browser
```
